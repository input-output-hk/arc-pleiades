Interactive Demo

Runs the actual library compiled to WebAssembly — real BLS12-381 field arithmetic, real Lagrange interpolation, real Guruswami-Sudan tracing.

WASM not loaded. Build with: cd arc-pleiades-wasm && wasm-pack build --target web --out-dir ../docs/pkg
Split — owner distributes shares; each guardian stores theirs
Verify — each guardian checks their share against public commitments (Feldman mode)
Reconstruct — t+1 guardians return stored shares
Trace — identify which guardians leaked (Traceable mode)
